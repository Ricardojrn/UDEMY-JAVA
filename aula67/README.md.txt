Nesta aula, aprendemos a criar uma aplicação simples em Java para manipular dados de um produto. Os principais conceitos abordados foram:

✅ Classes e Objetos: Criamos a classe Product para representar um produto com atributos e métodos.
✅ Encapsulamento básico: Trabalhamos diretamente com os atributos do objeto, alterando seus valores a partir do main.
✅ Entrada de dados com Scanner: Utilizamos Scanner para capturar informações do usuário.
✅ Métodos para manipulação de estoque: Implementamos funções para adicionar e remover produtos.
✅ Uso do toString: Personalizamos a exibição do objeto ao imprimir no console.

Este foi um passo importante para entender como modelar entidades do mundo real no código!
